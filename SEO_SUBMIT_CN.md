# 网站SEO提交 - 中文傻瓜式教程

## 第一步：提交到 Google Search Console（必做）

### 1.1 打开 Google Search Console
1. 打开浏览器，访问：https://search.google.com/search-console
2. 用你的 Google 账号登录（没有账号需要先注册）

### 1.2 添加网站
1. 点击「添加资源」按钮
2. 输入你的网站地址：`https://hjouy.github.io/ai-tools-directory/`
3. 点击「继续」

### 1.3 验证网站所有权（选择一种方式）

**方式一：HTML 文件验证（最简单，推荐）**
1. 选择「HTML 文件」标签
2. 下载验证文件 `googlea3f8e5d9c1e2b7.html`
3. 把这个文件放到你的网站根目录（和 index.html 同一个文件夹）
4. 点击「验证」

**方式二：HTML 标签验证（也很简单）**
1. 选择「HTML 标签」标签
2. 复制显示的 `<meta>` 标签代码
3. 把代码粘贴到 index.html 的 `<head>` 部分
4. 点击「验证」

### 1.4 提交 sitemap
1. 验证成功后，进入 Search Console 主界面
2. 左侧菜单点击「sitemap」
3. 在「添加 sitemap」输入框输入：`sitemap.xml`
4. 点击「提交」

---

## 第二步：提交到 Bing Webmaster Tools（推荐）

### 2.1 打开 Bing Webmaster
1. 访问：https://www.bing.com/webmasters
2. 用 Microsoft 账号登录

### 2.2 添加网站
1. 点击「添加网站」
2. 输入网站地址：`https://hjouy.github.io/ai-tools-directory/`
3. 点击「添加」

### 2.3 验证网站
1. 选择验证方式（推荐选择「自动验证」）
2. 按照提示操作

### 2.4 提交 sitemap
1. 进入网站管理后台
2. 点击「sitemap」
3. 输入：`sitemap.xml`
4. 点击「提交」

---

## 第三步：提交到百度搜索资源平台（国内用户推荐）

### 3.1 打开百度搜索资源平台
1. 访问：https://ziyuan.baidu.com
2. 用百度账号登录

### 3.2 添加网站
1. 点击「添加网站」
2. 输入网站地址
3. 选择网站类型

### 3.3 验证网站
1. 选择验证方式（推荐「HTML标签验证」）
2. 把百度提供的 meta 标签添加到 index.html
3. 点击验证

### 3.4 提交 sitemap
1. 验证成功后
2. 点击「链接提交」→「sitemap」
3. 输入你的 sitemap 地址

2. **Bing Webmaster**：在「配置」→「Sitemap」查看状态

## 常见问题

Q: 验证失败怎么办？
A: 确保验证文件已正确上传，或 meta 标签已添加到 index.html 正确位置

Q: 多久能看到效果？
A: Google 通常 1-2 周，Bing 3-5 天，百度 1 周左右

Q: 需要付费吗？
A: 完全免费！
